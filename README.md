# PW_final_kulakov

Мінімальний Playwright-проєкт для запуску end-to-end тестів.

## Як запустити

### 1. Клонувати репозиторій
```bash
git clone https://github.com/YaroslavKulakov/PW_final_kulakov.git
cd PW_final_kulakov
```

### 2. Встановити залежності
```bash
npm install
```

### 3. Запустити тести
```bash
npx playwright test
```

### 4. Переглянути HTML-звіт
```bash
npx playwright show-report
```

## Структура
```
tests/                 # Тестові файли
playwright.config.ts   # Конфігурація Playwright
package.json           # Залежності
```
